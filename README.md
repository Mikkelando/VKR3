## RICE2025-CH4

```bash
cd RICE13_pyomo/
python RICE2025_CH4.py -g
```

```bash
python RICE2025_CH4.py -g --nc False
```

```bash
python RICE2025_CH4.py -g --coop False
```

```bash
python RICE2025_CH4.py --coop False --nc False --map
```
